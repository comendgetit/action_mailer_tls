ActionMailerTls

Use GMail SMTP server to send emails in Rails ActionMailer

Example

Place code snippet in config/environment.rb

ActionMailer::Base.server_settings = { :address => "smtp.gmail.com", :port => 587, :domain => "mycompany.com", :authentication => :plain, :user_name => "username", :password => "password" }
